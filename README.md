# Hi, I'm Solar 👋

### Technical PM @ Microsoft CoreAI · Voice Agents Evaluation & Observability

I'm a **Technical Program Manager** on Microsoft's **CoreAI** org, where I work on making voice and speech AI **measurable** — the evaluation, benchmarking, and observability that show how good our voice agents really are, and where they fall short.

- 📊 **Evaluation & benchmarking** — accuracy, latency, and quality harnesses; competitive bake-offs
- 🔭 **Observability** — traffic, usage, and quality dashboards over large-scale telemetry
- 🤖 **Agentic tooling** — MCP servers, GitHub Copilot CLI skills, and natural-language data agents

- 📝 **Reports that ship** — closing the loop from eval round → decision → roadmap
